# Realm AI Chatbot

A modern AI-style chatbot built with **React**, **TypeScript**, **Tailwind CSS**, **Framer Motion**, and **Lottie animations**. Includes predefined questions, typing animations, Lottie mood panel, and a toggleable mini-game.
